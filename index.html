1. How to Install Minecraft

Choose Edition

• Java Edition – PC, mods, redstone
• Bedrock Edition – Mobile, Console, Windows

Download

Go to the official site → Buy or try Demo

minecraft.net/download

Launch

Install the launcher → Log in with Microsoft account → Hit Play!

3. First Day Survival (Most Important!)

  1. 1. Punch trees to get wood → Craft wooden tools
  2. 2. Make a Crafting Table and Wooden Pickaxe
  3. 3. Mine stone → Make Stone Tools
  4. 4. Find food (punch animals or eat apples)
  5. 5. Build a simple shelter before night falls!

4. Basic Crafting Recipes

Wood → 4 Wood Planks
4 Planks + 2 Sticks → Crafting Table
3 Wood Planks → Wooden Pickaxe
8 Cobblestone → Furnace
